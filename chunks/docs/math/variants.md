# Variants (math)

Alternate typefaces within formulas.

## Signature

`math.serif(content) -> content body content Required Positional`

## Examples

```typst
$ sans(A B C) $
```

```typst
$ frak(P) $
```

```typst
$ mono(x + y = z) $
```
