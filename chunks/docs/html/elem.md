# elem (html)

An HTML element that can contain Typst content.

## Signature

`html.elem(str, attrs: dictionary, nonecontent) -> content`

## Parameters

- `tag` (str, required): The element's tag.

- `attrs` (dictionary, optional): The element's HTML attributes.

- `body` (none | content, optional): The contents of the HTML element.
