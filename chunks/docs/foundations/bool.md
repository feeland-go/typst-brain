# bool (foundations)

A type with two states.

## Examples

```typst
#false \
#true \
#(1 < 2)
```

## Overview

Reference Foundations Boolean bool A type with two states. The boolean type has two values: true and false. It denotes whether something is active or enabled. Example #false \ #true \ #(1 < 2)
