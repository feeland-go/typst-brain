# Fletcher Overview

[Fletcher](https://github.com/Jollywatt/typst-fletcher) is a Typst package for drawing diagrams with nodes and arrows, built on top of [CeTZ](file:///media/feel/ubuntuehe/typst-brain/packages/cetz/_index.md).

Key Features:
- Flexible node placement (grid-based or manual).
- Powerful arrow/edge system with many heads and styles.
- Support for commutative diagrams in math mode.
- Flowcharts and state machines.
- Feynman diagrams.
