# Projects using CeTZ

CeTZ is a foundational package for many other Typst diagramming tools:

- [Fletcher](https://github.com/Jollywatt/typst-fletcher): Commutative diagrams.
- [Conchord](https://github.com/sitandr/conchord): Guitar fretboards.
- [Finite](https://github.com/jneug/typst-finite): Finite automata.
- [Chronos](https://git.kb28.ch/HEL/chronos): Sequence diagrams.

Tutorials and detailed scientific examples can be found at [janosh.github.io/diagrams](https://janosh.github.io/diagrams).
