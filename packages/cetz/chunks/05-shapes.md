# Drawing Shapes

CeTZ provides many built-in shapes.

### Line
`line((0, 0), (2, 2), ...)`

### Rectangle
`rect((0, 0), (2, 2))`

### Circle and Ellipse
`circle((0, 0), radius: 1)`
`ellipse((0, 0), radius: (1, 0.5))`

### Arc
`arc((0,0), start: 45deg, stop: 135deg, radius: 1)`

### Curve (Bézier)
`bezier((0,0), (2,2), (1,3))`
`curve((0,0), (1,1), (2,0), (3,1))`
