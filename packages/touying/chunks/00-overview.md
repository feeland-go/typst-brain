# Touying Overview

[Touying](https://github.com/touying-typ/touying) (投影 in chinese, /tóuyǐng/, meaning projection) is a user-friendly, powerful and efficient package for creating presentation slides in Typst.

Key Features:
- Split slides by headings.
- `#pause` and `#meanwhile` animations.
- Math Equation Animation.
- CeTZ and Fletcher Animations integration.
- Correct outline and bookmark (no duplicate and correct page number).
- Multiple themes (simple, university, metropolis, dewdrop, aqua).
- Semi-transparent cover mode.
- Speaker notes for dual-screen.
- Export to PPTX and HTML.
