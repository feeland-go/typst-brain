# touying (v0.6.2)

A powerful package for creating presentation slides in Typst.

## Import
```typst
#import "@preview/touying:0.6.2": *
```

## Example
```typst
= Section

== Subsection

=== First Slide

Hello, Touying!

=== Second Slide

Hello, Typst!
```
