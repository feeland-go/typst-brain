# Roots (math)

Square and non-square roots.

## Signature

`math.root(nonecontent, content) -> content index none or content Positional Settable`

## Examples

```typst
$ sqrt(3 - 2 sqrt(2)) = sqrt(2) - 1 $
$ root(3, x) $
```

```typst
$ root(3, x) $
```

```typst
$ sqrt(3 - 2 sqrt(2)) = sqrt(2) - 1 $
```
